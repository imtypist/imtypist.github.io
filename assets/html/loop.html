C++ 循环可视化 - 手动单步执行

👉 点击【单步执行】走一步 | 切换两层/三层循环

执行过程：
// 两层循环
for(int i=1;i<=2;i++){
    cout<<"外层 i="<<i<<endl;
    
    for(int j=1;j<=3;j++){
        cout<<"  内层 j="<<j<<endl;
    }
    
    cout<<"外层 i="<<i<<" 结束"<<endl;
}

// 三层循环
for(int i=1;i<=2;i++){
    cout<<"最外层 i="<<i<<endl;
    
    for(int j=1;j<=2;j++){
        cout<<"  中层 j="<<j<<endl;
        
        for(int k=1;k<=2;k++){
            cout<<"    内层 k="<<k<<endl;
        }
    }
    
    cout<<"最外层 i="<<i<<" 结束"<<endl;
}
